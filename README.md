# DesignCore Editorial

A high-performance editorial design system that merges Swiss International Style precision with a "Cyber-Corporate" digital vibrancy. Built with React 19, shadcn/ui v4, Base UI, and Tailwind CSS v4.

## Preview

🔗 **[Live Preview](https://jadewuu.github.io/ds-app/)**

## Design Tokens

| Token | Light | Dark |
|---|---|---|
| Primary | `#000000` | `#CCFF00` |
| Background | `#FFFFFF` | `#000000` |
| Border | `#E2E2E2` | `#333333` |
| Radius | `0px` | `0px` |
| Font | Inter Variable | Inter Variable |

## Tech Stack

- React 19
- TypeScript 6
- Tailwind CSS v4
- shadcn/ui v4.7
- Base UI primitives
- Lucide React icons
- Vite 8

## Getting Started

```bash
npm install
npm run dev
```

## Build
